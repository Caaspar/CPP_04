# CPP_04

CPP_04 est un projet du cursus 42.
Le but est d’approfondir la **programmation orientée objet** à travers l’**héritage**, le **polymorphisme** et les **classes abstraites** en C++.

## Compilation

```bash
make
```

## Contenu

* Implémentation de **classes abstraites et méthodes virtuelles pures**
* Utilisation du **polymorphisme** pour gérer des comportements dynamiques
* Gestion correcte de la **mémoire et des destructeurs virtuels**
